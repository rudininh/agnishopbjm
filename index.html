Agni Shop Banjarmasin API

Aplikasi internal untuk sinkronisasi stok & order dari berbagai marketplace: Shopee, Tokopedia, Lazada, TikTok Shop, dan Blibli.

Fitur utama:

  • Ambil daftar order dari Shopee
  • Update stok produk secara otomatis
  • Integrasi multi marketplace

Versi demo untuk keperluan verifikasi Shopee API.
